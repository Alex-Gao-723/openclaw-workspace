# 济州岛君悦酒店(Grand Hyatt Jeju)周边购物指南

> 📍 酒店地址：济州特别自治道济州市老连路12号 (12 Noyeon-ro, Jeju-si)
> 🏨 酒店位置优势：位于济州市中心，是济州岛最高、最大型的五星级酒店

---

## 🏢 一、免税店

### 1. 新罗免税店 (Shilla Duty Free)
- **地址**：济州特别自治道济州市老连路69号 (69, Noyeon-ro, Jeju-si)
- **距离酒店**：约700米，步行约8-10分钟
- **营业时间**：10:00-19:00 / 10:00-19:30（终年无休）
- **客服电话**：+82-2-2639-6000
- **特色**：
  - 济州岛规模最大的免税店，共3层楼
  - 唯一经销爱马仕、香奈儿、蒂芙尼、路易威登等顶级奢侈品
  - 拥有济州岛最大规模的奢华腕表卖场
  - 经销品牌300余种，包括化妆品、香水、服饰、手表、珠宝、箱包等
  - 1-2楼主要是奢侈品，2楼有韩国本土品牌MLB和ADLV，3楼是护肤品专柜
  - 韩国本土品牌可现场提货，国际品牌需机场提货

### 2. 乐天免税店 (Lotte Duty Free)
- **地址**：济州特别自治道济州市道令路83号 (83, Doryeong-ro, Jeju-si)
- **距离酒店**：约740米，步行约10分钟
- **营业时间**：10:00-18:00 / 10:00-19:00（全年无休）
- **特色**：
  - 位于乐天城市酒店内，占据1-3层楼
  - 汇集270多个奢侈品牌和济州特产
  - 首饰品牌丰富（尚美、麒麟、FRED等）
  - 与新罗免税店步行距离仅几分钟
  - 可乘机场大巴500路在GRACE HOTEL站下车

### 3. JDC济州机场免税店
- **位置**：济州国际机场内
- **距离酒店**：约2公里，车程约10分钟
- **特色**：
  - 销售烟酒、化妆品、香水、手表、箱包等300多个名牌
  - 一次性消费额度$600，年度允许消费6次
  - 适合最后补货

## 🛍️ 二、购物中心与超市

### 1. E-mart 易买得超市
- **距离酒店**：步行约5分钟（酒店隔壁/附近）
- **营业时间**：10:00-23:00（每月第二周五/第四周六停业）
- **特色**：韩国大型综合超市，食品、日用品齐全

### 2. 乐天超市 (Lotte Mart)
- **距离酒店**：步行约5-10分钟
- **特色**：韩国本土大型超市，可购买伴手礼和日用品

### 3. Olive Young 美妆店
- **距离酒店**：步行约5-10分钟（酒店对面/附近）
- **特色**：韩国最大药妆连锁店，化妆品、护肤品、保健品齐全

### 4. 便利店
- **GS25、CU、7-Eleven**
- **距离酒店**：酒店3楼有24小时7-Eleven，楼下多家便利店
- **特色**：零食、饮料、日常用品、香蕉牛奶等

## 🚶 三、特色购物街区

### 1. 七星路购物街 (Chilsungro Shopping Street)
- **韩语名称**：칠성로 쇼핑거리
- **地址**：济州市一徒一洞一带
- **距离酒店**：约3-4公里，车程约10分钟
- **营业时间**：大部分店铺11:00-21:00/22:00（各店铺略有不同）
- **特色**：
  - 济州岛最具代表性的商圈，被称为"济州的明洞"
  - 汇集国际运动品牌（耐克、阿迪达斯等），价格约为国内半价
  - 品牌集合店：ABC-MART、S-Market等鞋类折扣店
  - 周边有山地川音乐喷泉、塔洞广场
  - 邻近黑猪肉一条街、生鱼片一条街
  - 适合购买运动服饰和鞋子

### 2. 中央地下商场 (Jungang Underground Shopping Center)
- **地址**：济州市一徒1洞1425-3号
- **距离酒店**：约3-4公里
- **营业时间**：10:00-22:00
- **特色**：
  - 连接七星路与东门市场的地下购物街
  - 约280多家店铺，以中低价服装、鞋类、饰品、化妆品为主
  - 设施现代，有婴儿车租借服务
  - 头顶有拱廊，防晒防雨

### 3. 宝健路/莲洞步行街 (Bao Jian Road / Nuwemaru Street)
- **距离酒店**：约700米-1公里，步行约10分钟
- **特色**：
  - 新济州最繁华的商业街
  - 各种小店、餐厅、咖啡厅、换钱所
  - 运动品牌专卖店较多

### 4. 东门传统市场 (Dongmun Traditional Market)
- **地址**：济州市二徒一洞1432
- **距离酒店**：约3-4公里
- **营业时间**：
  - 周一至周五：06:00-20:00
  - 周六：09:00-12:00
  - 周日休市
- **特色**：
  - 济州岛规模最大的传统市场
  - 新鲜农渔产品、土特产、日常用品
  - 可品尝糖饼、炒年糕、金鱼饼等小吃
  - 推荐购买：脱水蔬菜、带鱼、鲅鱼、柑橘、紫菜、泡菜等

### 5. 神话街
- **距离酒店**：约3公里
- **特色**：有ABC MART等鞋类折扣店

## 💡 五、购物实用贴士

### 退税信息
- 在标有"Tax Free"的店铺购物满30,000韩元可退税
- 购物时索取退税单，出境时在机场海关盖章确认后托运
- 免税店商品已免税，无需再退税

### 免税店购物须知
1. **购物时间**：建议提前1-2天购买，国际品牌需时间送至机场
2. **提货方式**：
   - 韩国本土品牌（MLB、ADLV、后、雪花秀等）：可现场提货
   - 国际品牌（香奈儿、LV等）：需在机场提货
3. **返点优惠**：自由行游客可找返点公司办理金卡，享受额外返点（化妆品约25-30%返点）
4. **支付方式**：支持支付宝、微信支付、银联卡、信用卡

### 交通建议
| 目的地 | 距离酒店 | 交通方式 |
|--------|----------|----------|
| 新罗免税店 | 700米 | 步行 |
| 乐天免税店 | 740米 | 步行 |
| 七星路/东门市场 | 3-4公里 | 出租车约10分钟 |
| 济州机场 | 2公里 | 出租车约10分钟 |

### 营业时间提醒
- 济州岛商店一般**10:00-11:00**才开门
- 晚上通常营业到**21:00-22:00**
- 周末部分店铺可能休息或缩短营业时间
- 传统市场周日可能休市

### 携带限制（入境中国）
- ❌ **禁止携带**：新鲜水果（橘子等）、生鲜肉类、海鲜
- ✅ **可以携带**：真空包装的食品、加工零食、巧克力、茶叶、保健品
- ⚠️ 建议提前了解中国海关最新规定

## 📋 六、推荐购物路线

```
酒店 → 新罗免税店 → 乐天免税店 → 宝健路 → 酒店
```

```
酒店 → 东门市场（上午） → 七星路购物街 → 中央地下商场 → 黑猪肉街晚餐 → 酒店
```
