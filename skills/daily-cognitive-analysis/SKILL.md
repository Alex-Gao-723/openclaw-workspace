# 每日认知分析报告技能

## 用途

分析录音卡笔记（语音备忘录转录），生成结构化的每日认知分析报告。帮助用户从日常记录中提取价值、发现模式、跟踪行动项。

## 适用场景

- 每日录音卡笔记自动分析
- 周期性笔记回顾（周/月/季度）
- 关键决策前的历史信息检索
- 个人知识管理与沉淀

## 使用方法

### 1. 分析单日笔记

```bash
python3 skills/daily-cognitive-analysis/analyze.py --date 2026-04-07
```

### 2. 分析日期范围

```bash
python3 skills/daily-cognitive-analysis/analyze.py --start-date 2026-04-01 --end-date 2026-04-07
```

### 3. 分析昨天（默认行为）

```bash
python3 skills/daily-cognitive-analysis/analyze.py
```

### 4. 发送飞书通知

```bash
python3 skills/daily-cognitive-analysis/analyze.py --date 2026-04-07 --notify
```

## 输出格式

报告包含以下结构化章节：

1. **今日总览** - 3-5条最重要的总结
2. **关键事件与决策** - 事件、人物、结论、原文依据
3. **行动项清单** - 表格形式的待办事项
4. **重要洞察** - 值得长期保存的观点和想法
5. **重复出现的主题** - 跨笔记的重复话题
6. **风险与未决问题** - 分歧、障碍、潜在风险
7. **明天最值得关注的5件事** - 优先级建议
8. **可归档标签** - 便于检索的关键词

## 核心分析原则

### 提取优先级
- ✅ 高价值信息：决策、洞察、行动项、风险
- ❌ 过滤噪音：寒暄、重复表达、无效内容

### 区分信息类型
- **事实** - 明确发生的事情
- **判断** - 基于事实的观点
- **猜测** - 不确定的推测（需标注"不确定"）

### 认知演进追踪
- 识别观点的变化轨迹
- 标记关键转折点
- 连接相关但分散的记录

## 文件位置

- 笔记源文件：`/root/.openclaw/workspace/录音卡笔记/YYYY-MM-DD.md`
- 分析报告：`/root/.openclaw/workspace/reports/daily-report-YYYY-MM-DD.md`

## 配置

编辑 `config.json` 可自定义：
- 笔记源目录
- 报告输出目录
- 飞书通知接收人
- 分析模型参数

## 定时任务建议

```cron
# 每日早上8点分析前一天的录音卡笔记
0 8 * * * cd /root/.openclaw/workspace && python3 skills/daily-cognitive-analysis/analyze.py --notify
```

## 分析方法论

### 1. 信息分层
```
原始记录 → 事实提取 → 洞察提炼 → 行动转化
```

### 2. 模式识别
- **时间模式** - 特定主题出现的时间规律
- **语义模式** - 重复出现的概念和关键词
- **行为模式** - 用户的思考和行为习惯

### 3. 价值判断
- **紧急重要矩阵** - 区分处理优先级
- **可执行性评估** - 判断行动项的可落地性
- **知识价值** - 是否值得进入长期知识库

## 注意事项

1. **不编造** - 只分析已记录的内容
2. **标注不确定** - 信息不清晰时明确标注
3. **重构而非复述** - 帮助用户理清混乱的表达
4. **连接而非孤立** - 发现笔记间的关联
5. **行动导向** - 最终目标是产生可执行的行动

## 依赖

- Python 3.8+
- kimi-coding API (用于智能分析)
- 飞书 API (用于通知发送)

## 扩展
